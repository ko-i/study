---
##### 1. putchar(getchar())是一个有效地表达式，它实现什么功能？getchar(putchar())是否也是有效地表达式？
> 第一个是有效地，它将输入流中的字符再放回输出流中。第二个不行，因为putchar()需要参数，getchar()不需要参数。
---
##### 2. 下面的语句分别完成什么任务？

> * 把控制字符响铃放入输出流中。
> * 把换行符放入输出流中。
> * 把空格放入输出流中。
