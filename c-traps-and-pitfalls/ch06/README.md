``` c
static int num1, num2;
#define max(a,b) (num1 = (a); num2 = (b); num1 > num2 ? num1 : num2)
```

#### 2. 本章第1节中提到的“表达式”
``` c
(x)((x)-1)
```
#### 能否成为一个合法的C表达式？
* 能，当#define x （某类型） 时，比如`#define x int`，这时候表达式变为`(int((int)-1))`等价于把-1强制转换为int两次。
